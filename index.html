About the Project

CVL-CDSAML is an innovative digital humanities project that combines computational linguistics with historical language analysis. We are creating the first comprehensive valency lexicon that tracks how verb argument structures have evolved across multiple ancient and medieval languages over more than two millennia.

Computational Methods

Using state-of-the-art NLP tools and machine learning algorithms to analyze historical texts, extract valency patterns, and visualize language change over time.

Historical Depth

Spanning from ancient to medieval texts, covering crucial periods of language change across multiple linguistic traditions.

Open Access

All our corpora, tools, and research outputs are freely available under open licenses, fostering collaboration and advancing the field of digital historical linguistics.

Languages Under Investigation

  • Ancient and medieval language varieties spanning over 2000 years
  • Focus on diachronic changes in valency patterns
  • Comparative analysis across multiple language families

Project Objectives

Main Research Questions

  1. Linguistic Genealogy vs. Contact: How do we distinguish between inherited valency patterns and those resulting from language contact?
  2. Diachronic Pathways: What are the typical trajectories of valency change across different language families?
  3. Computational Modeling: Can we predict valency changes using machine learning models trained on historical data?
  4. Cross-linguistic Patterns: Are there universal tendencies in how argument structures evolve over time?

Expected Outcomes

  • Comprehensive digital corpus with 1+ million annotated tokens
  • Interactive online valency lexicon covering 5,000+ verbs
  • Open-source computational tools for historical linguistics
  • New theoretical insights into language change mechanisms
  • Educational materials for university courses and summer schools

Work Packages

WP1 Months 1-3

Project Management

Establishment of project infrastructure, team coordination, and research protocols.

Key Activities:
  • Set up computational infrastructure and version control systems
  • Establish data management protocols following FAIR principles
  • Create project website and communication channels
  • Organize kick-off meeting with all partners
WP2 Months 2-10

Selection & Analysis of Texts

Corpus compilation and initial linguistic analysis of selected ancient and medieval texts.

Text Selection Criteria:
  • Representative texts from each linguistic period
  • Balance between literary and documentary sources
  • Texts with existing digital editions when possible
  • Coverage of different text types and registers
Corpus Size Targets:
  • Total target: 1.2+ million tokens
  • Balanced representation across all language varieties
  • Multiple time periods for each tradition
WP3 Months 4-15

Annotation of Valency Features

Systematic annotation of valency patterns in relation to language contact and genealogy.

Annotation Framework:
  • Universal Dependencies enhanced with valency-specific tags
  • Semantic role labeling for argument structures
  • Contact-induced change markers
  • Genealogical relationship indicators
Tools & Methods:
  • Custom annotation interface based on WebAnno
  • Inter-annotator agreement protocols
  • Machine learning-assisted annotation for efficiency
  • Regular annotation workshops and training
WP4 Months 7-18

Automatic Extraction & PROIEL Integration

Development of computational tools for valency extraction and integration with existing resources.

Technical Development:
  • Python-based valency extraction pipeline
  • Integration with PROIEL treebank infrastructure
  • API for querying valency patterns
  • Visualization tools for diachronic changes
Expected Outputs:
  • Open-source extraction toolkit
  • REST API for data access
  • Interactive web interface
  • Documentation and tutorials
WP5 Months 15-21

Statistical Analysis & Visualization

Quantitative analysis of valency patterns and creation of the contrastive lexicon.

Analysis Methods:
  • Diachronic frequency analysis of valency patterns
  • Contact-induced vs. genealogical change detection
  • Statistical modeling of change trajectories
  • Cross-linguistic pattern comparison
Visualization Components:
  • Interactive timeline visualizations
  • Network graphs for genealogical relationships
  • Heat maps for contact intensity
  • Searchable online lexicon interface
WP6 Months 19-24

Dissemination & Sustainability

Publication of results, community building, and ensuring long-term accessibility.

Dissemination Activities:
  • Two international workshops (Athens & Oslo)
  • Conference presentations at major venues
  • Open access journal publications
  • Integration with Naxos Diachronic Linguistic School
  • Collaboration with GlossaContact Lab
  • CIVIS BIP course: "Diachronic Linguistic in the 21st Century"
Educational Integration:
  • Materials for MA Program in English, Linguistics and Translation (Director: N. Lavidas)
  • PhD dissertation support for Maria Episkopou and Georgia Stavrianopoulou
  • Summer school modules at Naxos
