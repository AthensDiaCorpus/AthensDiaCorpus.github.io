Project Objectives

Main Research Questions

  1. Linguistic Genealogy vs. Contact: How do we distinguish between inherited valency patterns and those resulting from language contact?
  2. Diachronic Pathways: What are the typical trajectories of valency change across different language families?
  3. Computational Modeling: Can we predict valency changes using machine learning models trained on historical data?
  4. Cross-linguistic Patterns: Are there universal tendencies in how argument structures evolve over time?

Expected Outcomes

  • Comprehensive digital corpus with 1+ million annotated tokens
  • Interactive online valency lexicon covering 5,000+ verbs
  • Open-source computational tools for historical linguistics
  • New theoretical insights into language change mechanisms
  • Educational materials for university courses and summer schools

Research Timeline

Phase 1 Initial Stage

Project Management

Establishment of project infrastructure, team coordination, and research protocols.

Key Activities:

  • Set up computational infrastructure and servers
  • Establish data management protocols
  • Create project website and documentation
  • Initialize corpus collection procedures
Phase 2 Corpus Development

Diachronic Corpus Compilation

Collection and digitization of texts across three millennia in Greek, Romance, and Germanic languages.

Corpus Targets:

  • Ancient Greek: Homer to Byzantine texts (500,000+ tokens)
  • Romance: Latin to modern varieties (300,000+ tokens)
  • Germanic: Gothic to modern languages (200,000+ tokens)
  • XML/TEI encoding for all texts
Phase 3 Annotation Process

Linguistic Annotation

Comprehensive annotation of valency patterns, argument structures, and syntactic features.

Annotation Layers:

  • Morphological tagging (part-of-speech, case, number)
  • Syntactic dependencies (Universal Dependencies framework)
  • Semantic roles (agent, patient, experiencer)
  • Valency frames and alternations
Phase 4 Tool Development

Computational Tool Creation

Development of specialized software for diachronic linguistic analysis.

Tools & Platforms:

  • Web-based corpus query interface
  • Valency pattern visualization tools
  • Machine learning models for change prediction
  • API for computational access
Phase 5 Analysis Stage

Statistical and Contrastive Analysis

In-depth analysis of diachronic patterns and cross-linguistic comparisons.

Research Focus:

  • Quantitative analysis of valency changes
  • Cross-family pattern comparison
  • Contact-induced vs. inherited changes
  • Theoretical model development
Phase 6 Dissemination

Publication and Outreach

Sharing results through publications, conferences, and educational initiatives.

Outputs:

  • Open-access valency lexicon database
  • Peer-reviewed publications in major journals
  • International conference presentations
  • Educational workshops and summer schools

Educational Initiatives

Naxos Diachronic Linguistic School

Annual summer school on the island of Naxos focusing on computational methods in historical linguistics. Participants learn corpus annotation, statistical analysis, and visualization techniques.

GlossaContact Lab

Research laboratory focusing on language contact phenomena and their computational modeling. State-of-the-art facilities for corpus analysis and experimental linguistics.

CIVIS BIP Course

"Diachronic Linguistics in the 21st Century" - Intensive course for European students combining traditional philology with cutting-edge computational methods.

MA Program

Master's in English, Linguistics and Translation (Director: Prof. Lavidas). Specialized track in computational historical linguistics with hands-on corpus work.
