About the Project

Project Objectives

Main Research Questions

  1. Linguistic Genealogy vs. Contact: How do we distinguish between inherited valency patterns and those resulting from language contact?
  2. Diachronic Pathways: What are the typical trajectories of valency change across different language families?
  3. Computational Modeling: Can we predict valency changes using machine learning models trained on historical data?
  4. Cross-linguistic Patterns: Are there universal tendencies in how argument structures evolve over time?

Expected Outcomes

  • Comprehensive digital corpus with 1+ million annotated tokens from Homer to today
  • Interactive online valency lexicon covering 5,000+ verbs
  • Open-source computational tools for historical linguistics
  • Diachronic retranslations database with open access
  • Integration with Penn-Helsinki and PROIEL standards
  • New theoretical insights into language change mechanisms
  • Educational materials for university courses and summer schools

Collaborative Tools & Standards

The project employs established computational linguistics infrastructure including Penn-Helsinki parsing standards, PROIEL treebank architecture, and open access tools for diachronic analysis. All resources, including diachronic retranslations and annotated corpora, will be freely available to the research community.

Research Timeline

Phase 1 Project Setup

Infrastructure Development

Establishing computational infrastructure, team coordination protocols, and initial corpus collection procedures.

Phase 2 Corpus Building

Text Collection & Digitization

Gathering texts from Greek (Homer to today), Romance (Latin to modern), and Germanic traditions. Target: 1+ million tokens. Special focus on diachronic retranslations and open access materials.

Phase 3 Annotation

Linguistic Annotation

Comprehensive annotation including morphology, syntax (Universal Dependencies), semantic roles, and valency frames.

Phase 4 Tool Development

Software Creation

Building web interfaces, query tools, visualization platforms, and machine learning models for pattern detection.

Phase 5 Analysis

Comparative Analysis

Statistical analysis of diachronic patterns, cross-linguistic comparisons, and theoretical model development.

Phase 6 Dissemination

Publication & Outreach

Open-access database release, journal publications, conference presentations, and educational workshops.
